## 개요
* **프로젝트 팀** : HomeSecurity
* **프로젝트 취지** : 토이 프로젝트를 통한 API 활용 및 프론트엔드의 보편적인 라이브러리 활용 연습
* **진행 기간** : 2023.12.02 - 2023.12.20
* **한줄 소개** : 유튜브 영상 댓글 좋아요 수가 많은 순서대로 정렬하여 사용자에게 제공
* **서비스 URL** : [YouTubeLikedComment](https://youtube-liked-comment.vercel.app/)
